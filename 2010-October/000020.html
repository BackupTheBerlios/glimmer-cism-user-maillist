Hi Jonny,

I'm coping this answer to the Glimmer-CISM user mailing list: I would 
suggest using this in future, as it means you can draw on a much larger 
range of expertise.

I'm afraid that GLINT hasn't been improved to deal with rotated grids at 
this point - I think it would be a rather big job!

As for forcing Glimmer with data on the correct grid, you would probably 
be best to look at the EIS test example in the glimmer-tests release:

https://developer.berlios.de/project/showfiles.php?group_id=10995&release_id=16552

This forces a fennoscandian run with a parameterized climate, but it 
could be easily adapted to use one prescribed from file. So, you'd be 
using GLIDE directly, rather than through GLINT.

Hope that helps,
