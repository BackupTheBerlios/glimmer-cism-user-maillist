Hi Kevin,
you need to check out the source code from subversion, i.e.
svn co \ 
http://svn.berlios.de/svnroot/repos/glimmer-cism/glimmer-cism2/trunk

This will produce a new directory called trunk which contains all the 
source code. Before you can run configure you will need to run ./bootstrap.

For info on subversion, have a look at
http://svnbook.red-bean.com/
