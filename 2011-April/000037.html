Hi Flo-

The version of Glimmer you are using has a configure that doesn’t allow for some of AIX’s specific needs out of the box.  Its not seeing the –DHAVE_CONFIG. All the –D options need to have a –WF,-C!  In front. Same deal with the other error message you sent. Here is my install script for a different AIX machine with a different version of Glimmer you can play with:

FC="mpxlf90_r -O2 -qsuffix=cpp=F90 -qalias_size=107374182 -WF,-C!" \
F90="mpxlf90_r -O2 -qsuffix=cpp=F90 -qalias_size=107374182 -WF,-C!" \
F77="mpxlf_r -O2  -qsuffix=cpp=F90 -qalias_size=107374182 -WF,-C!" \
CC="mpcc_r -O2 -qcpluscmt" \
CXX="mpCC_r -O2" \
LIBS="-lC_r" \
--with-netcdf=/contrib/netcdf-3.6.2 \
--with-lapack=-lessl \
--with-blas=-lblas  \

Kate Evans
